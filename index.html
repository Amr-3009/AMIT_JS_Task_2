Array Methods:

Basic Array Methods

Array Length

It returns the length (size) of an array

Example

const fruits = ["Banana", "Orange", "Apple", "Mango"];

let size = fruits.length;

In this case it returns 4

toString() Method

The toString() method converts all elements of an array into a string

Example

const arr = [1, 2, 3];
const str = arr.toString();
console.log(str); // Output: 1,2,3

at() Method

The at() method is used to access an element from an array using its index and if that index does not exist in the array then it will return undefined.

Example

const arr = [1, 2, 3];
let firstElement = arr.at(0);
console.log(firstElement); // Output: 1

join() Method

The join() method joins all elements of an array into a single string.
like toString() but with join we can specifiy the separator

Example

const arr = ['I', 'love', 'coding'];
const result = arr.join('_');
console.log(result); // Output: I_love_coding

pop() Method

The pop() method removes the last element from an array.

Example

const arr = [5, 8, 9];
const removedItem = arr.pop();
console.log(removedItem); // Output: 9 console.log(arr); // Output: [5, 8]

push() Method

The push() method adds a new element at the end of an array

Example

const arr = [1, 2, 3];
arr.push(4, 5, 6);
console.log(arr); // Output: [1, 2, 3, 4, 5, 6]

shift() Method

The shift() method is used to remove the first element from an array. It returns the value that was originally in the first position. It also shifts all other elements to a lower index

Example

const arr = [1, 2, 3];
const removedItem = arr.shift();
console.log(removedItem); // Output: 1

unshift() Method

The unshift() method is used to add one or more elements to the start of an array and return the new length of the array. Unlike push(), it does not need to create a new array; instead, it modifies the existing array by adding the new items to its beginning. It also unshifts older elements giving them a higher index

Example

const arr = [2, 3, 4];
arr.unshift(1);
console.log(arr); // Output: [1, 2, 3, 4]

delete() Method

The delete operator can be used with arrays. But to remove an item from an array we should use pop() or shift(). As it leaves undefined holes in the array

Example

const arr = ["apple", "banana", "mango"];
delete arr[0];
console.log(arr); // Output: ['undefined', 'banana', 'mango']

concat() Method

The concat() method is used to merge two or more arrays.

Example

let arr1 = [1, 2, 3];
let arr2 = [4, 5, 6];
let result = arr1.concat(arr2)
console.log(result); // Output: [1, 2, 3, 4, 5, 6]

In case of merging/concatation of more than 2 arrays
arr1.concat(arr2, arr3);

copyWithin() Method

The copyWithin() method is used to copy specified elements from an array to another position within the same array. It returns the modified array it overwrites the existing value, doesn't add items to the array and it doesn't change the array length

Example

let arr = [5, 8, 9, 3, 6, 7];
arr.copyWithin(1, 4) note: it'll copy (6) arr[4] to (8) arr[1]
console.log(arr); // Output: [5, 6, 9, 3, 6, 7]

splice() Method

splice() method is used to add new items to an array

Example

let numbers = [1, 2, 3, 4, 5];
numbers.splice(2, 0, 10, 11);
console.log(numbers); // Output: [1, 2, 10, 11, 3, 4, 5]

The first parameter (2) defines the position where new elements should be added (spliced in).
The second parameter (0) defines how many elements should be removed.

Deleting with splice()

let numbers = [1, 2, 3, 4, 5];
numbers.splice(0, 1);
console.log(numbers); // Output: [2, 3, 4, 5]

Giving us a safe way to delete without leaving undefined holes in our array

toSpliced() Method

Pretty much the same as splice() method but it provides a safer way by not altering the orignal array as the new toSpliced() method creates a new array while the old splice() alters the original array

slice() Method

The slice() method slices out a piece of an array into a new array

Example

const fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
const citrus = fruits.slice(1); note: starting from index[1] it made a new array
console.log(citrus); // Output: ["Orange", "Lemon", "Apple","Mango"]
it can take multiple parameters such as slice(1, 3)
which means starting from index[1] up to but not including index[3]
meaning we select elements[1, 2]
